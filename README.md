# Penta Organizasyon

Professional event management company website based in Antalya, Turkey.

## Overview

Penta Organizasyon provides comprehensive event management services including:
- Corporate event management
- Part-time staff supply
- Stage, decoration, sound & lighting
- VIP transfer services

**Website:** [pentaorganizasyon.com](https://www.pentaorganizasyon.com)

## Tech Stack

- **HTML5** - Semantic markup with SEO optimization
- **CSS3** - Modular architecture with CSS variables
- **JavaScript** - Vanilla JS with module pattern
- **3D Graphics** - GLB models with model-viewer
- **PWA** - Service worker for offline support

## Features

- Responsive design for all devices
- Dark/Light theme support
- Multi-language support (Turkish/English)
- 3D animated elements
- Progressive Web App (PWA)
- SEO optimized with structured data

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/penta_organization.git
   ```

2. Open `index.html` in your browser or serve with a local server:
   ```bash
   npx serve .
   ```
